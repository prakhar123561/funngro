## 2026 SEO Implementation Notes

### On-Page Essentials
- Single, clear H1 per page; structured H2/H3 hierarchy
- Unique title tags and meta descriptions per page
- High-intent internal links to signup, project categories, and FAQs
- Schema suggestions:
  - `FAQPage`
  - `Organization`
  - `Review`/`AggregateRating` (where valid)

### Content Intent Coverage
- **Teen page intent:** aspiration + first income + low barrier to start
- **Business page intent:** ROI + operational simplicity + reliability

### Performance/UX Targets
- Mobile-first rendering
- Core Web Vitals readiness (LCP, INP, CLS)
- Optimized visual assets (WebP/AVIF), lazy loading for wall-of-fame and case-study media

## Messaging Summary
- **Teen narrative:** "I can start now, earn safely, and build confidence."
- **Business narrative:** "I can scale digital work quickly with vetted young talent at lower cost."

Both pages should preserve a unified Funngro brand voice while adapting tone:
- Teen page: energetic, aspirational, community-driven
- Business page: professional, metrics-backed, trust-led
